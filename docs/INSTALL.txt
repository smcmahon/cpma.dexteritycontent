cpma.dexteritycontent Installation
----------------------------------

To install cpma.dexteritycontent using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``cpma.dexteritycontent`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        cpma.dexteritycontent
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
